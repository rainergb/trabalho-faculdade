# Portfólio - Versão HTML/CSS/JS

Esta é uma versão simplificada do portfólio em HTML, CSS e JavaScript puro, sem frameworks como Next.js.

## Estrutura
- `index.html`: Página principal
- `styles.css`: Estilos adicionais
- `script.js`: Funcionalidades JavaScript

## Como executar
Abra o arquivo `index.html` em um navegador web.
